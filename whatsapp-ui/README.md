# WhatsApp Chat List UI - React Recreation

## Overview
A recreation of the WhatsApp Chat List screen built with React + Vite.

## Component Structure
App
├── Navbar        → Top green bar with title and icons
├── SearchBar     → Input field to search chats
└── ChatItem      → Reusable chat card (avatar, name, message, time)

## Features
- Reusable ChatItem component using props
- Responsive design for mobile, tablet and desktop
- WhatsApp-style green theme

## How to Run
npm install
npm run dev

## Tech Stack
- React.js
- Vite
- CSS
